# ✨ Features

* **Fixed Navigation Bar** — Smooth-scroll navigation between sections
* **Typing Animation** — Dynamic rotating headline
* **Scroll-Reveal Animations** — Smooth entrance animations while scrolling
* **Active Navigation Highlighting**
* **LeetCode Awards Sidebar** — Custom hexagonal achievement badges
* **Projects Showcase** — Detailed projects with tech stacks and GitHub links
* **Developer Dashboard**

  * Codolio profile
  * LeetCode statistics
  * GitHub contribution graph
* **Tech Stack Section** — Skill icons for languages, frameworks, databases, and tools
* **Resume Download Button**
* **Fully Responsive Design** — Works on mobile, tablet, and desktop
