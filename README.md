# 🌐 Ankit Kumar — Portfolio Website

A personal portfolio website built with pure HTML & CSS showcasing my skills, projects, hackathon achievements, and coding profiles.

🔗 **Live Preview:** [ankit-kumar-developer-122.github.io/Portfolio](https://ankit-kumar-developer-122.github.io/Portfolio) *(enable GitHub Pages to activate)*

---

## ✨ Features

- **Fixed Navigation Bar** — Smooth-scroll links to all sections (Home, Projects, Hackathons, Resume, Contact)
- **Logo & Profile Photo** — Personal photo used as both the hero section avatar and navbar logo
- **Typing Animation** — Rotating animated text headline in the hero section
- **Scroll-Reveal Animations** — Sections fade in as you scroll
- **Active Nav Highlighting** — Current section highlighted in the navbar while scrolling
- **LeetCode Awards Sidebar** — Hexagon-shaped achievement badges (8 awards)
- **Projects Section** — 6 projects with full descriptions, tech stacks, and GitHub links
- **Dashboard** — Live Codolio stats, LeetCode card, and GitHub contribution graph
- **Tech Stack** — Icons for all languages, frameworks, databases, and tools
- **Download Resume Button** — Direct link to Google Drive resume
- **Responsive Design** — Works on mobile, tablet, and desktop
